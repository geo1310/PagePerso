Site Web pour un Restaurant

Projet Web dans le Domaine de la Restauration

L'objectif ? Permettre à ce restaurant de gagner en visibilité et de proposer son menu de Pizzas aux utilisateurs.

Le client étant novice dans les nouvelles technologies, il aura la possibilité de modifier facilement son menu en ligne grace à une interface d'administration sur-mesure.

Ce projet est simple mais fonctionnel.

Technologies Utilisées : HTML, CSS, DJANGO, PYTHON, HEROKU
